# depend

 * `python3`
 * `v2ray-core` [v2ray-core](https://github.com/v2fly/v2ray-core)

# install v2ray-core

download latest `v2ray-core` package

```
cd v2ray
cp -f v2ray ~/.local/bin
cp -f geoip.dat ~/.local/bin
cp -f geosite.dat ~/.local/bin
cp -f config.json ~/.local/etc
```

# usage

```
./v2ray_cli.py
```
